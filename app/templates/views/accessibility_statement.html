Accessibility statement

This accessibility statement applies to the GOV.UK Emergency Alerts service, https://admin.emergency-alerts.service.gov.uk.

The Emergency Alerts service is run by the GOV.UK Emergency Alerts team at the Government Digital Service (GDS). It’s designed to be accessible by as many people as possible. You should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 300% without the text spilling off the screen
  • listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

The text should be clear and simple to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

How accessible this website is

Parts of this website are not fully accessible:

  • validation errors may be confusing for screen reader users
  • the ‘copy to clipboard’ button does not work with some screen readers
  • if a team adds template folders, screen readers may announce the page heading in a confusing way
  • Windows users cannot view our icons in high contrast mode
  • screen reader users may find it hard to expand or collapse groups of checkboxes

Feedback and contact information

If you need any part of this service in a different format such as large print, audio recording or braille, contact the GDS Emergency Alerts team. We’ll consider your request and get back to you within 3 working days.

Reporting accessibility problems with this website

We’re always looking to improve the accessibility of this website. If you find any problems not listed on this page or think we’re not meeting accessibility requirements, contact the GDS Emergency Alerts team.

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 .

If you're not happy with how we respond to your complaint contact the Equality Advisory and Support Service (EASS).

Technical information about accessibility of this website

GDS is committed to making its website accessible in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 .

GOV.UK Emergency Alerts is partially compliant with the Web Content Accessibility Guidelines (WCAG) version 2.1 AA standard.

Non-accessible content

The content listed below is non-accessible for the following reasons.

Non-compliance with the WCAG version 2.1 AA standard

  1. Some icons do not appear when using high contrast mode on Windows. This fails success criterion 1.4.8: visual presentation. We plan to fix this in the future.

Problems with using the interface

  1. Our validation errors cause problems for screen readers. This means it can be hard to know when an error has occurred or what to do next. We have fixed the majority of these issues. We are planning to fix the remaining validation errors in the immediate future.
  2. The JAWS screen reader announces template folder page headings as multiple sentences. We are investigating this issue and plan to fix it in the future.
  3. Expanding and collapsing groups of checkboxes using a screen reader can be confusing. We plan to fix this by December 2023.

How we tested this service

In May 2022, the Digital Accessibility Centre (DAC) conducted an audit of parts of the website used to author and manage alerts. The service was tested again by DAC in June 2023, including an audit of the alerts that are sent out and the web pages users see when they send an alert.

The test was based on the most common tasks that GOV.UK Emergency Alerts users need to complete. This included testing a sample of pages with common user interface components from multiple user roles including Author, Approver and Platform Administrator.

What we’re doing to improve accessibility

Following the audit conducted by DAC in June 2023, we have identified content which fails to meet the Web Content Accessibility Guidelines version 2.1 AA standard. We plan to fix these issues in the future.

Preparation of this accessibility statement

This statement was published on 13 July 2023. It was last reviewed on 6 July 2023. It will be reviewed again on 13 August 2023.
